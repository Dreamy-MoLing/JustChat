# UI Redesign Design Spec

日期: 2026-05-29
状态: 草案
范围: JustTalk Flutter 应用全页面 UI 重设计

## 设计目标

将 JustTalk 从"方形无圆角"风格升级为现代圆角风格，提升用户体验和视觉吸引力。

## 配色方案

### 主色系（相近色，用于页面和组件）

| 颜色 | Hex | 用途 |
|------|-----|------|
| 蓝 | `#3B82F6` | 欢迎页动画、聊天页面渐变 |
| 青（主色） | `#0D9488` | AppBar、按钮、气泡图标、强调 |
| 浅青 | `#5EEAD4` | 渐变终点、头像渐变 |
| 绿 | `#10B981` | 欢迎页动画、在线状态、成功提示 |

### 互补色（用于强调和通知）

| 颜色 | Hex | 用途 |
|------|-----|------|
| 琥珀 | `#F59E0B` | 步骤卡数字背景、通知强调 |
| 暖黄 | `#FDE68A` | 步骤卡渐变、警告提示 |
| 暖底色 | `#FFF8E1` | 通知卡片背景 |

### 中性色

| 颜色 | Hex | 用途 |

## 圆角规范

告别 `BorderRadius.zero`，统一使用圆角：

| 元素 | 圆角值 |
|------|--------|
| 卡片 | 14-16px |
| 头像 | 12-14px |
| 按钮 | 16px |
| 输入框 | 14px |
| 底边栏图标容器 | 14-18px |
| 消息气泡 | 16px（对方：左下 4px；自己：右下 4px） |
| 底部弹窗 | 24px（顶部） |
| 步骤卡数字 | 10px |

## 欢迎页动画

### 流程

1. 页面初始：浅灰白背景 `#F8FAFB`
2. 三个柔和光晕（蓝、青、绿）从不同位置淡入
3. 光晕向中心聚拢，逐渐融合
4. 光晕消失，中心偏上位置出现单色气泡图标（纯 `#0D9488`，无渐变、无背景）
5. 气泡图标下方淡入 "JustChat" 品牌文字，两者保持间距，互不遮挡
6. 停留片刻后过渡到引导步骤页

### 技术要点

- 光晕使用 `radial-gradient` + `filter: blur(40-50px)`
- 气泡图标为单色 `#0D9488`，圆角 20px，白色聊天气泡 SVG，位于中心偏上
- "JustChat" 文字位于气泡图标下方，保持间距，`z-index` 置于顶层，`pointer-events: none`
- 动画时长约 3 秒

## 引导步骤页

欢迎页动画结束后过渡到此页面。

- 背景：浅渐变 `#F0FDFA → #ECFDF5 → #F0FDFA`
- 顶部：气泡图标（单色青）+ "欢迎使用 JustChat" + 副标题
- 步骤卡：白色卡片，左侧带互补色竖条强调
  - 步骤 1（青）：设置昵称
  - 步骤 2（蓝）：扫描二维码
  - 步骤 3（绿）：开始聊天
- 底部：渐变按钮 "开始设置"

## 底边栏导航

替换原有的侧边栏/Drawer，改为底部导航栏。

### 三个标签

| 位置 | 名称 | 图标 | 说明 |
|------|------|------|------|
| 左 | 圈子 | ◉ | 社交动态信息流（避嫌"朋友圈"） |
| 中 | 聊天 | 💬 气泡图标 | 凸起居中，主色强调，单色青 |
| 右 | 个人 | ☰ | 设置、个人信息、自定义项 |

### 凸起图标设计

选中项的图标容器向上凸起，视觉突出：

- 宽度 48px，高度 48px，圆角 16px
- 背景色：主色 `#0D9488`（选中时）
- 白色图标
- 阴影：`0 4px 16px rgba(13,148,136,0.3)`
- 向上凸起约 18-22px
- 聊天 tab 使用气泡 SVG 图标，其他 tab 使用对应图标

### 选中/未选中状态

- **选中项**：图标容器凸起 + 主色 `#0D9488` 背景 + 白色图标 + 主色文字 w600
- **未选中项**：无凸起 + 灰色图标 `#64748B` + 灰色文字
- **默认标签**：打开软件默认进入"聊天"tab

## 页面列表

### 1. 欢迎页 (WelcomePage)

- 全屏动画 → 引导步骤 → "开始设置"按钮
- 首次启动展示，完成后不再显示

### 2. 首页 (HomePage) — 底边栏"聊天"tab

底边栏中间的"聊天"标签页，是应用的主页面。

- AppBar：渐变背景 `#0D9488 → #0FAAA0`，标题"聊天"
- 搜索栏：白色圆角卡片
- 联系人/聊天列表：白色圆角卡片，带头像、在线状态、最后消息、未读 badge
- FAB：圆角 18px，渐变背景，打开连接方式选择
- 点击联系人 → 进入 ChatPage

### 3. 圈子页 (CirclePage) — 底边栏"圈子"tab

- 本次仅实现空壳页面 + 占位提示
- 后续版本实现社交动态信息流

### 4. 个人页 (ProfilePage) — 底边栏"个人"tab

- 头像卡片：大头像 + 昵称 + Peer ID
- 设置列表：圆角分组卡片
  - 通知设置 → NotificationsPage
  - 信令服务器 → SettingsPage
  - 使用帮助 → InfoPage

### 5. 聊天页 (ChatPage)

- AppBar：渐变背景，返回按钮、头像、在线状态
- 连接状态栏：颜色编码（绿=已连接、黄=连接中、红=失败）
- 消息气泡：
  - 自己：渐变 `#0D9488 → #0FAAA0`，白色文字，右对齐
  - 对方：白色背景，深色文字，左对齐
  - 圆角 16px，尖角侧 4px
- 输入栏：白色背景，圆角输入框，渐变发送按钮

### 6. 设置页 (SettingsPage) — 个人页子页面

- 三个分区：个人信息、连接、偏好
- 圆角卡片分组

### 7. 通知页 (NotificationsPage)

- 互补色强调未读通知
- 圆角卡片列表
- 空状态：居中铃铛图标

### 8. 二维码扫描页 (QrScannerPage)

- 全屏扫描器
- 圆角扫描框引导

### 9. 回答二维码页 (AnswerQrPage)

- 显示生成的回答二维码
- 圆角卡片容器

## 响应式布局

- 移动端（< 600px）：单列 + 底边栏
- 桌面端（≥ 600px）：保留底边栏，聊天 tab 内部改为左侧联系人列表 300px + 右侧嵌入 ChatPage
